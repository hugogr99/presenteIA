# [presenteIA](https://hugogr99-presente-ia.hf.space/) 🎁 

O **presenteIA** é um motor de recomendação inteligente desenvolvido para transformar descrições subjetivas em sugestões reais de presentes, utilizando IA Generativa e busca semântica.

### Tecnologias e Conceitos Aplicados
* **LLM & IA Generativa:** Utilização da API do **Groq (Llama 3)** para análise de perfil e extração de características semânticas.
* **NLP & Embeddings:** Transformação de perfis em vetores matemáticos utilizando a biblioteca `sentence-transformers` (`paraphrase-multilingual-MiniLM-L12-v2`).
* **Busca Semântica:** Cálculo de **Similaridade de Cosseno** para encontrar o match perfeito entre o presenteado e a base de produtos.
* **Reinforcement Learning:** Implementação de um sistema de feedback baseado em **Q-learning**, onde o algoritmo aprende com as avaliações positivas e negativas dos usuários.
* **Data Engineering:** Pipeline de extração e consolidação de dados de grandes e-commerces (Amazon, Mercado Livre, Renner, Boticário) utilizando **PySpark** e Databricks.

### Contexto
Este projeto foi desenvolvido como uma aplicação prática dos conceitos estudados no MBA em Data Science e Analytics na **USP/Esalq**, unindo Engenharia de Dados, Ciência de Dados e Inteligência Artificial para resolver um problema cotidiano.
